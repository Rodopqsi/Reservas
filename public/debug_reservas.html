Debug del Sistema de Reservas

Rutas de Prueba:

1. Auto Login (Profesor) 2. Listado de Reservas 3. Ver Reserva ID 1 4. Ver Reserva ID 2 5. Debug Reserva ID 1 6. Calendario

Información:

El error "Route [reservas.destroy] not defined" puede deberse a:

  • Caché de rutas o vistas persistente
  • Archivos temporales en el navegador
  • Algún componente que aún referencia la ruta antigua

Pasos para reproducir:

  1. Hacer login con el botón "Auto Login"
  2. Ir al listado de reservas
  3. Hacer click en "Ver" en alguna reserva
  4. Observar si aparece el error
